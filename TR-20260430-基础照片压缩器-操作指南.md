## 二、痛点深度研究报告

### 2.1 痛点来源与用户原话

#### 痛点1：批量压缩崩溃（严重度：极高）

**用户原话（Reddit r/ios）**:
> "basically over the years ive taken too many screenshots and i need a way to get RID OF DUPLICATE but EVEN MORE IMPORTANTLY compress the screenshots in one batch ideally in the background. obviously I doubt theres a free app that does this but at least not a pay monthly one"
> — OneFisherman9541

> "JUST TO BE SUPER CLEAR I WANT TO DO ENTIRE SCREESHOT FOLDER IN ONE GO, NOT ONE BY ONE."

> "Did you ever find an app like that?" — 5个月后仍无解决方案

**App Store差评（Compressor: Photos & Pictures）**:
> "This app works great, it does what it supposed to do. However, the app crashes over 10 images."

**分析**: 现有应用在处理超过10-50张照片时频繁崩溃，用户需要真正稳定的批量处理能力。

#### 痛点2：订阅疲劳（严重度：极高）

**用户原话（Reddit r/setupapp）**:
> "I built an iOS image compression app because I was frustrated with subscription-based tools"
> — 独立开发者也认为订阅模式不合理

**用户原话（Reddit r/ios）**:
> "They force you into a subscription." — 对强制订阅的强烈不满

**分析**: 照片压缩是简单工具功能，用户强烈抗拒月度订阅。一次性买断或免费+一次性解锁是用户期望的模式。

#### 痛点3：截图占用巨大空间（严重度：高）

**用户原话（Reddit）**:
> "over the years ive taken too many screenshots" — 截图累积是普遍问题

**分析**: iPhone截图默认保存为PNG格式，单张可达1-5MB。用户累积数百张截图占用数GB空间，但现有应用不专门针对截图优化。

#### 痛点4：隐私担忧（严重度：高）

**用户原话（Reddit r/ios）**:
> "does it compress photos? is it safe to use, privacy"

**分析**: 用户担心照片上传到服务器。100%本地处理、零数据收集是刚需。

#### 痛点5：质量不可控（严重度：中高）

**App Store差评**:
> "Doesn't work with screenshot/PNG files" — PNG格式支持差
> "Issues with saving compressed photos" — 保存问题

**分析**: 用户需要精确控制压缩质量，且需要前后对比确认质量可接受。

#### 痛点6：无法自动删除原图（严重度：中）

**App Store差评（iOS 26）**:
> "Fails to delete original photo on iOS 26" — 新系统兼容性问题

**分析**: 压缩后需要可靠地删除原图才能真正释放空间，但很多应用在这方面有bug。

6. **无Share Extension**: 大多数不支持从其他App直接压缩分享

#### 5.3.1 压缩引擎 (CompressionEngine.swift)

```swift
import Foundation
import Photos
import MediaToolSwift
import SwiftData

enum CompressionQuality: Double, CaseIterable {
    case low = 0.3
    case medium = 0.5
    case high = 0.7
    case best = 0.85
    
    var displayName: String {
        switch self {
        case .low: return "Low"
        case .medium: return "Medium"
        case .high: return "High"
        case .best: return "Best"
        }
    }
    
    var estimatedReduction: String {
        switch self {
        case .low: return "~85%"
        case .medium: return "~70%"
        case .high: return "~50%"
        case .best: return "~30%"
        }
    }
}

enum OutputFormat: String, CaseIterable {
    case jpeg = "JPEG"
    case heic = "HEIC"
    case png = "PNG"
    
    var mediaFormat: ImageFormat {
        switch self {
        case .jpeg: return .jpeg
        case .heic: return .heic
        case .png: return .png
        }
    }
}

struct CompressionConfig {
    var quality: CompressionQuality = .high
    var outputFormat: OutputFormat = .jpeg
    var maxWidth: CGFloat? = nil
    var maxHeight: CGFloat? = nil
    var stripMetadata: Bool = false
    var deleteOriginal: Bool = false
}

@MainActor
class CompressionEngine {
    
    private let batchProcessor = BatchProcessor()
    
    func compressSingle(
        sourceURL: URL,
        config: CompressionConfig,
        progressHandler: @escaping (Double) -> Void
    ) async throws -> CompressionResult {
        let sourceAttributes = try FileManager.default.attributesOfItem(
            atPath: sourceURL.path
        )
        let originalSize = sourceAttributes[.size] as? Int64 ?? 0
        
        let destinationURL = generateDestinationURL(
            source: sourceURL,
            format: config.outputFormat
        )
        
        let sizeSetting: ImageSize
        if let maxWidth = config.maxWidth, let maxHeight = config.maxHeight {
            sizeSetting = .fit(CGSize(width: maxWidth, height: maxHeight))
        } else if let maxWidth = config.maxWidth {
            sizeSetting = .fit(CGSize(width: maxWidth, height: maxWidth * 10))
        } else {
            sizeSetting = .fit(CGSize(width: 4096, height: 4096))
        }
        
        let info = try await ImageTool.convert(
            source: sourceURL,
            destination: destinationURL,
            settings: .init(
                format: config.outputFormat.mediaFormat,
                size: sizeSetting,
                quality: config.quality.rawValue
            )
        )
        
        let compressedAttributes = try FileManager.default.attributesOfItem(
            atPath: destinationURL.path
        )
        let compressedSize = compressedAttributes[.size] as? Int64 ?? 0
        
        let reduction = originalSize > 0
            ? Double(originalSize - compressedSize) / Double(originalSize) * 100
            : 0
        
        if config.deleteOriginal {
            try? FileManager.default.removeItem(at: sourceURL)
        }
        
        return CompressionResult(
            originalURL: sourceURL,
            compressedURL: destinationURL,
            originalSize: originalSize,
            compressedSize: compressedSize,
            reductionPercentage: reduction,
            format: config.outputFormat
        )
    }
    
    func compressBatch(
        sourceURLs: [URL],
        config: CompressionConfig,
        progressHandler: @escaping (Int, Int, Double) -> Void
    ) async throws -> [CompressionResult] {
        return try await batchProcessor.processBatch(
            items: sourceURLs,
            config: config,
            compressAction: { url, cfg, itemProgress in
                try await self.compressSingle(
                    sourceURL: url,
                    config: cfg,
                    progressHandler: itemProgress
                )
            },
            batchProgressHandler: progressHandler
        )
    }
    
    private func generateDestinationURL(source: URL, format: OutputFormat) -> URL {
        let directory = FileManager.default.temporaryDirectory
            .appendingPathComponent("ShrinkPic", isDirectory: true)
        try? FileManager.default.createDirectory(
            at: directory, withIntermediateDirectories: true
        )
        
        let ext: String
        switch format {
        case .jpeg: ext = "jpg"
        case .heic: ext = "heic"
        case .png: ext = "png"
        }
        
        let filename = source.deletingPathExtension()
            .lastPathComponent + "_shrunk"
        return directory.appendingPathComponent("\(filename).\(ext)")
    }
}
```

#### 5.3.2 批量处理器 (BatchProcessor.swift)

```swift
import Foundation

actor BatchProcessor {
    
    private var isCancelled = false
    
    func processBatch<T, R>(
        items: [T],
        config: CompressionConfig,
        compressAction: (T, CompressionConfig, @escaping (Double) -> Void) async throws -> R,
        batchProgressHandler: @escaping (Int, Int, Double) -> Void
    ) async throws -> [R] {
        isCancelled = false
        var results: [R] = []
        let total = items.count
        
        for (index, item) in items.enumerated() {
            if isCancelled { break }
            
            let result = try await compressAction(item, config) { itemProgress in
                let overallProgress = (Double(index) + itemProgress) / Double(total)
                batchProgressHandler(index + 1, total, overallProgress)
            }
            results.append(result)
            
            batchProgressHandler(index + 1, total, Double(index + 1) / Double(total))
            
            try await Task.sleep(nanoseconds: 10_000_000)
        }
        
        return results
    }
    
    func cancel() {
        isCancelled = true
    }
}
```

#### 5.3.3 存储分析器 (StorageAnalyzer.swift)

```swift
import Foundation
import Photos

struct StorageCategory: Identifiable {
    let id = UUID()
    let name: String
    let icon: String
    let size: Int64
    let count: Int
    let color: String
}

@MainActor
class StorageAnalyzer {
    
    func analyzePhotoLibrary() async throws -> [StorageCategory] {
        let options = PHFetchOptions()
        let allAssets = PHAsset.fetchAssets(with: .image, options: options)
        
        var screenshots: (size: Int64, count: Int) = (0, 0)
        var camera: (size: Int64, count: Int) = (0, 0)
        var downloads: (size: Int64, count: Int) = (0, 0)
        var other: (size: Int64, count: Int) = (0, 0)
        
        for index in 0..<allAssets.count {
            let asset = allAssets[index]
            let resources = PHAssetResource.assetResources(for: asset)
            
            var estimatedSize: Int64 = 0
            for resource in resources {
                if let fileSize = resource.value(forKey: "fileSize") as? Int64 {
                    estimatedSize += fileSize
                }
            }
            
            if asset.mediaSubtypes.contains(.screenShot) {
                screenshots.size += estimatedSize
                screenshots.count += 1
            } else if asset.sourceType == .camera {
                camera.size += estimatedSize
                camera.count += 1
            } else {
                other.size += estimatedSize
                other.count += 1
            }
        }
        
        return [
            StorageCategory(
                name: "Screenshots",
                icon: "camera.viewfinder",
                size: screenshots.size,
                count: screenshots.count,
                color: "#FF6B6B"
            ),
            StorageCategory(
                name: "Camera Photos",
                icon: "photo.fill",
                size: camera.size,
                count: camera.count,
                color: "#4ECDC4"
            ),
            StorageCategory(
                name: "Other Photos",
                icon: "photo.on.rectangle.angled",
                size: other.size,
                count: other.count,
                color: "#45B7D1"
            )
        ]
    }
    
    func formatBytes(_ bytes: Int64) -> String {
        let formatter = ByteCountFormatter()
        formatter.allowedUnits = [.useGB, .useMB]
        formatter.countStyle = .file
        return formatter.string(fromByteCount: bytes)
    }
}
```

```swift
@Model
final class CompressionResult {
    var originalURLPath: String
    var compressedURLPath: String
    var originalSize: Int64
    var compressedSize: Int64
    var reductionPercentage: Double
    var format: String
    var timestamp: Date
    
    init(
        originalURL: URL,

        originalSize: Int64,
        compressedSize: Int64,
        reductionPercentage: Double,
        format: OutputFormat
    ) {
        self.originalURLPath = originalURL.path
        self.compressedURLPath = compressedURL.path
        self.originalSize = originalSize
        self.compressedSize = compressedSize
        self.reductionPercentage = reductionPercentage
        self.format = format.rawValue
        self.timestamp = Date()
    }
    
    var savedBytes: Int64 {
        originalSize - compressedSize
    }

    var formattedCompressedSize: String {
```

### 5.4 编写规则

1. **Swift Concurrency**: 所有异步操作使用 `async/await`，UI更新使用 `@MainActor`
2. **SwiftUI声明式**: 所有视图使用SwiftUI，避免UIKit混编
3. **SwiftData**: iOS 17+使用SwiftData替代CoreData
4. **零第三方追踪**: 不集成任何Analytics SDK，所有统计本地存储
5. **内存管理**: 批量处理时使用 `autoreleasepool` 和分块处理，避免内存溢出
6. **错误处理**: 使用Swift原生 `throws/try/catch`，不使用Result类型
7. **命名规范**: 遵循Swift API Design Guidelines，变量名清晰描述功能
8. **文件组织**: 按功能模块分组，每个文件不超过300行

### 9.4 价格心理学分析

**$2.99定价依据**:
1. **心理定价**: $2.99 < $3.00，属于"两块多"心理区间
2. **价值锚定**: 竞品订阅$4.99/周 = $259.48/年，ShrinkPic $2.99终身 = 节省$256
3. **冲动消费区间**: $0.99-$4.99是App Store冲动消费黄金区间
4. **功能价值比**: 释放数GB存储空间 vs $2.99，用户感知价值极高
5. **iCloud替代**: iCloud 50GB = $0.99/月，ShrinkPic一次性$2.99 = 永久节省iCloud费用

### 9.5 免费层策略详解

**为什么是5张/天而不是3张或10张**:
- 3张/天: 太少，用户无法充分体验，转化率低
- 10张/天: 太多，用户可能不需要Pro，付费转化率低
- **5张/天**: 平衡点 — 足够体验质量，但批量需求驱动Pro购买

**免费层功能设计原则**:
- 让用户能完整体验核心压缩功能（质量OK）
- 让用户感受到批量处理的必要性（痛点驱动）
- 不在免费层放广告（避免负面体验）
- 不在免费层加水印（避免质量担忧）
